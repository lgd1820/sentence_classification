# sentence_classification


개요
------------------
CNN 기반 문장 분류 기법을 이용한 인텐트 분류

인텐트
- 사용자의 입력에 따라 분류되는 단위
- ex) “날씨 어때?”, “내일 날씨 어때” -> “날씨 정보 조회”  

워드 임베딩 및 문장 분류
------------------
단어를 N 차원 백터로 변경시켜 학습이 가능한 숫자 형태로 변경
